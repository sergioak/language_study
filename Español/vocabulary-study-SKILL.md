# Vocabulary Study Skill

## 触发条件
用户给出一个西班牙语单词（可以是单独的词、图片中的词、或直接说"学xxx"），要求生成学习卡片。

## 输出位置
- 文件夹：`~/estudiar_español/`
- 命名：`{单词}.md`（小写，原形）

## 卡片结构模板

### 1. 📖 基础信息
用表格展示：
- 单词（加粗）
- 音标（西班牙语音标，如 [im.porˈtan.te]）
- 词性（adj./m./f./v./adv./prep. 等）
- 核心释义（2-4个中文释义，顿号分隔）

### 2. 🧠 记忆辅助
- **联想记忆**：拆解词根词缀，编一个简短有逻辑的联想故事
- **词源补充**：来自哪个拉丁语/希腊语词根，字面意思如何引申

### 3. ✏️ 核心用法
按词性归纳2-4种主要用法场景，每种给出：
- 用法名称（如"作定语"、"与ser搭配"、"固定句型"）
- 简短说明

### 4. 📝 例句模块（5-10句）
- 按用法分类，每种用法2-5个例句
- 每句格式：**西班牙语句子** + 换行 + 中文翻译
- 覆盖日常口语和书面表达

### 5. 📋 习题模块（10道）
分三部分：
- **选择题（核心用法）**：3-4道，考查词义、语法搭配、句型
- **近义词辨析**：3-4道，考查与近义词的区别和选择
- **反义词与否定用法**：2-3道，考查反义词、否定句式

格式：
```
**题号.** 题目内容
- A) 选项A
- B) 选项B
- C) 选项C
- D) 选项D
```

附参考答案表格：
| 题号 | 答案 | 解析 |

### 6. 🔗 相关词汇
表格形式：
| 词汇 | 含义 | 关系 |
关系列：近义 / 反义 / 派生名词 / 派生动词 / 同根词

## 注意事项
- 音标使用西班牙语标准音标（非英语音标）
- 动词给出原形（infinitivo），不同时态变化在例句中体现
- 如果用户发的是图片，先用 vision_analyze 识别内容，再按模板生成
- 如果用户只给了一个词（无图片），直接查证后按模板生成
- 形容词注意阴阳性变化说明（如 importante 无变化，但 bueno/buena 需说明）
- 动词注意是否为不规则动词，如有重要变位在用法中说明

## 示例
用户输入：`importante`
输出文件：`~/estudiar_español/importante.md`
